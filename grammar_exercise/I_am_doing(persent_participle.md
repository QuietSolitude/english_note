# I am doing(persent participle)

She's eating.    She isn't reading.    
It's raining.    The sun isn't shining.    
They're running.    They aren't walking.    

現在進行時的形式為：    
------------------------------------------------------------------------
am/is/are + doing/eating/running/writing等。     

|      |          |      |
| ---- | -------- | ---- |
| I    | am(not)  |      |
| he   |          |      |
| she  | is(not)  | -ing |
| it   |          |      |
| we   |          |      |
| you  | are(not) |      |
| they |          |      |
